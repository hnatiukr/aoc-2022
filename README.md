# Solving AoC 2022 problems in TypeScript

### Install

```console
make
```

### Build

```console
make build
```

### Run

> `day` is an argument. Pass a valid value in the format `01`, `08`, `12`, `25`, etc., which corresponds to one of 25 days

```console
make run day=01
```
